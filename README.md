# Test analyzer

This is a simple sentiment analyzer using NLTK created for learning purposes.

## Features

- Defines mood of the text
- User can add his own dicitionaries


## Usage
1. Run "analyzer.py".
2. Choose the way to analyze the text.
3. Recieve results.
